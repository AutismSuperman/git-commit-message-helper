# Git Commit Message Helper GitHub Pages

This branch contains the static website for GitHub Pages.

## Files

- `index.html`: product homepage.
- `usage.html`: installation, configuration, screenshots, and usage guide.
- `styles.css`: shared responsive styling.
- `app.js`: navigation and screenshot preview behavior.
- `assets/screenshots`: screenshots and GIF copied from the plugin documentation.
- `assets/brand`: plugin icon assets.

## GitHub Pages Settings

In the GitHub repository, enable Pages with:

- Source: `Deploy from a branch`
- Branch: `gh-pages`
- Folder: `/ (root)`

Expected site URL:

`https://www.fulinlin.com/git-commit-message-helper/`

The GitHub Pages project URL also redirects from:

`https://autismsuperman.github.io/git-commit-message-helper/`
